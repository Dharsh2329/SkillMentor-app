# DreamVault 🛩️

DreamVault is a simple web application that helps users organize their goals and stay focused on achieving them.

## Features

* Interactive splash screen
* User login/register interface
* Goal selection system

  * Placement Preparation
  * Develop New Skill
  * Learn Something New
    
* Task dashboard
* Responsive and modern UI design

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your web browser.
3. Enter your name and start planning your goals.

## Purpose

DreamVault is designed to motivate users to track their dreams, create goals, and stay focused on personal and professional growth.
